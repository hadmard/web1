# 会员系统 3.0 开发说明

会员系统不是广告售卖系统。

本系统定位为：
- 企业结构参与系统
- 行业共建机制
- 权威分层机制
- 数据结构支撑系统

## 已落地核心能力

1. 三级会员类型
- `memberType`: `enterprise_basic | personal | enterprise_advanced`
- 支持到期字段 `memberTypeExpiresAt`
- 支持自动降级：会话读取时检测到期并降级为 `personal`
- 支持权重字段 `rankingWeight`

2. 内容审核状态
- 内容状态统一字段 `status`: `draft | pending | approved | rejected`
- 企业侧内容提交默认 `pending`
- 管理端支持状态更新（审核通过/驳回）

3. 企业内容结构化关联字段
- 资讯支持：栏目、标签、关联词条、关联标准、关联品牌、是否同步主站
- 图库支持：标签、是否同步主站

4. 权限分层（member_type + role）
- 个人会员：不可发布企业资讯/图库，可提交标准反馈
- 企业基础会员：可发布资讯/图库（限额），默认待审核
- 企业高级会员：发布数量不受限，可主动关联标准与词条
- 管理员：审核、发布、状态流转

5. 安全基础
- 登录失败次数统计 + 短时锁定（防暴力破解）
- 会话输出含 memberType 与 rankingWeight，便于前端按钮级权限控制

## 数据关联目标
系统通过结构化字段对以下栏目实现关联基础：
- 整木词库
- 整木标准
- 整木数据
- 整木品牌
- 整木评选
- 整木图库
- 整木资讯
