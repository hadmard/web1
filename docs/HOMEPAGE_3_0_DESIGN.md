# 中华整木网 3.0 首页设计说明

## 核心定位
- 中华整木网定位为：中国整木定制行业知识基础设施平台。
- 首页角色不是“资讯流首页”，而是“七大结构入口页”。
- 首页目标：
  - 强结构：栏目与数据关系清晰、可被 AI 与搜索引擎抓取。
  - 强信任：通过标准、评选、报告、合作单位建立权威感。
  - 强会员联动：企业会员可进入品牌、图库、资讯、标准共建等结构链路。

## 首页结构顺序
- 顶部权威定位区（品牌 + 关键词导航 + 结构化搜索）
- 七大栏目结构入口区（首页核心）
- 热门需求区
- 权威背书区
- 品牌展示区
- 数据与标准区
- 图库标签区
- 会员入驻引导区

## 结构化规范
- 所有栏目主标题使用 H2。
- 子栏目标签使用 H3。
- 内容条目均链接至独立 URL。
- 搜索支持分类：品牌、标准、术语、榜单、数据、图库标签。
- 标签体系使用全站统一 tags 结构（style/craft/space/product/enterprise）。

## 视觉与交互
- 风格：极简、结构化、权威、去门户化、去广告化。
- 交互：延续 Apple 风格动效（reveal、glass、parallax、route transition）。
- 动画一致性：同类模块统一 reveal + interactive-lift 节奏，减少风格割裂。
